# Skills Library

This folder is intended to hold downloadable skills and code recipes. 
Whenever you receive external instructions or patterns, place them in this folder.
AI Coding agents will reference these skills and pick suitable ones for the current task.

To use:
1. Download or paste a skill file (e.g., `stripe-integration.md`) into this folder.
2. Ensure the rules and conventions in the file align with the project in `.ai/constraints.md`.
3. The AI agent will read this folder to augment its local context.
